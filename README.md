# YouTube Bookmark Chrome Extension

A Chrome extension that allows users to bookmark timestamps in YouTube videos for quick access later.

## Features
- Save timestamps while watching videos<br>
- View saved bookmarks in popup<br>
- Jump to bookmarked time instantly<br>
- Lightweight and fast<br>

## Tech Stack
- JavaScript<br>
- Chrome Extension APIs<br>
- HTML/CSS<br>

## Installation
1. Clone repo<br>
2. Open Chrome → Extensions<br>
3. Enable Developer Mode<br>
4. Load Unpacked<br>
5. Select the extension folder<br>
